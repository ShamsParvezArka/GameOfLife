### Pre-requirements
Compiler: <a href="https://gcc.gnu.org/"><b>gcc</b></a> <br>
Graphics library: <a href="https://raylib.com"><b>raylib</b></a>

### Compile
    chmod +x build.sh
    ./build.sh
### Run
    ./bin/gol       #to run in cli mode
    ./bin/raygol    #to run in gui mode
